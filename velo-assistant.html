Features

AI that fits your desktop

Glass Chat Bubbles

User messages in steel-blue glass. Claude responses in deep charcoal glass. The Velo aesthetic throughout.

Streaming Responses

Claude's response streams token-by-token using the Anthropic streaming API. No waiting for the full response.

Chat History

Conversations persist to disk. Browse previous sessions from the history sidebar. Search your entire chat history.

Native GTK4

Built with GTK4 and Rust. No Electron, no browser, no WebView. 12 MB. Starts in under 200ms.

Markdown Rendering

Claude's responses render as formatted markdown — code blocks with syntax highlighting, lists, headers, tables.

Local API Key

Your Anthropic API key stays on your machine, passed directly to the API. No cloud service, no middleman.
