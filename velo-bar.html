What's inside

Every instrument deliberate

Live Clock

Displays current time and date. Clicking opens a mini calendar popup for quick date reference.

CPU & RAM

Live system stats with smooth animation. Values update every 2 seconds with a subtle pulse.

Battery

Current charge percent, charging state, and estimated time to full or empty on click.

Network

WiFi signal indicator with click-to-expand speed popup showing live download and upload rates.

Bluetooth

Toggle Bluetooth on and off with a single click. Glows blue when active with connected device list.

Volume

Current volume percent. Click to reveal a precision drag slider synced to the OSD overlay.

Media Controls

Displays current track from any MPRIS source. Click to play/pause, scroll to seek.

Power Menu

Shutdown, restart, suspend, lock — all accessible from the power pill with a single click.

GTK4 Layer Shell

Renders above all windows using gtk4-layer-shell. Works on any Wayland compositor that supports the protocol.
