Features

Git commits, elevated

Claude API

Sends your staged diff to Claude Sonnet with a structured prompt. Returns a concise, accurate commit message.

Streaming

The message streams token-by-token into the glass panel — you see the response build in real time.

One-Click Commit

Approve the generated message and click Commit. Velo runs git commit with it — no copy-paste.

Diff Preview

Shows your staged diff in the panel so you can review it before generating. Syntax-highlighted by file type.

Edit Before Commit

The message is fully editable before committing. You have the final say — Claude just does the heavy lifting.

GTK4 Glass Window

Floats above your editor in the Velo glass aesthetic. Dismiss with Escape when done — no terminal needed.
