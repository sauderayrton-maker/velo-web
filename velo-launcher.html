Velo / Launcher

App Launcher

Fuzzy search
for zero mouse

A single glass search pill docks at the bottom centre of your display. Type anything — apps, recent files, commands — and results float above it as you type. Keyboard only.

⌕ ↑↓ navigate · ↵ launch

Features

Fast. Precise. Keyboard-first.

Fuzzy Matching

Smith-Waterman fuzzy search ranks results by quality. Type "vrox" to find "Firefox". Typo-tolerant.

Keyboard Nav

Arrow keys to navigate. Enter to launch. Escape to dismiss. Every interaction reachable without a mouse.

App Icons

Displays the system icon for every application using the XDG icon theme. Scales to HiDPI.

Instant Launch

No startup delay. The daemon pre-scans your .desktop files on login so the pill opens in <80ms.

Custom Commands

Prefix with ":" to run arbitrary shell commands. Results pipe into a read-only output panel.

Layer Shell

Rendered with gtk4-layer-shell at OVERLAY priority. Appears above all windows including fullscreen games.
